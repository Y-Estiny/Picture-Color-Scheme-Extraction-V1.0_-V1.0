# 图片主色提取色卡工具

一个纯前端的图片配色提取工具。用户选择或拖入图片后，工具会在浏览器本地分析图片主色，展示每个颜色的 HEX、RGB 和画面占比，并生成可保存、可导出的配色方案。

## 功能特点

- 本地图片分析，不上传用户图片
- 支持点击上传和拖拽上传
- 可选择提取 3 到 9 个主色
- 显示主色占比综合色条
- 显示每个颜色的 HEX、RGB 和百分比
- 支持一键复制单个 HEX 色值
- 支持导出 PNG 色卡图
- 支持导出多种设计软件和通用交换格式

## 导出格式

| 格式 | 用途 |
| --- | --- |
| `.png` | 可视化色卡图，适合展示、分享和归档 |
| `.ase` | Adobe 色板文件，适合 Photoshop、Illustrator、InDesign 等 |
| `.stylx` | ArcGIS Pro 配色迁移清单 |
| `qgis_style.xml` | QGIS 样式文件 |
| `office_theme.thmx` | Microsoft PowerPoint / Word / Excel 主题文件 |
| `.json` | 通用结构化配色数据 |
| `.css` | CSS 变量 |
| `.scss` | SCSS 变量和颜色映射 |
| `.csv` | 表格、GIS、数据可视化等通用交换 |
| `.zip` | 一次性打包全部软件适配格式 |

> 注意：ArcGIS Pro 的原生 `.stylx` 是样式库数据库格式。当前导出的 `.stylx` 文件用于迁移颜色信息和 CIM RGBColor 数据，建议在 ArcGIS Pro 中先用小样本验证导入流程。

## 使用方式

直接打开 `index.html` 即可使用。

如果需要通过本地服务预览，可以在项目目录运行：

```bash
python -m http.server 4173 --bind 127.0.0.1
```

然后访问：

```text
http://127.0.0.1:4173/index.html
```

## 技术说明

- 无需后端和数据库
- 不依赖打包工具
- 图片颜色分析基于浏览器 `canvas`
- 主色聚类使用 Lab 色彩空间距离，相比简单 RGB 分组更接近视觉感知
- 下载文件均由浏览器端即时生成

## 适合场景

- 从参考图中快速提取品牌或视觉配色
- 为 Photoshop、QGIS、Office、前端页面等生成可复用色卡
- 为设计汇报、视觉分析、地图制图和数据可视化准备配色素材
- 建立图片到设计系统颜色变量的轻量工作流
